# Pandas Data Analysis Projects

This repository contains multiple data analysis projects implemented using **Python** and the powerful data manipulation library **Pandas**. The main goal of these projects is to demonstrate practical skills in data cleaning, preprocessing, exploratory data analysis (EDA), and extracting meaningful insights from real-world datasets.

These projects showcase how structured and unstructured datasets can be processed efficiently to identify patterns, trends, and relationships. Various data manipulation techniques such as filtering, grouping, aggregation, and statistical analysis were used to understand the datasets and generate useful insights.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib

## Projects Included

### 1. Customer Churn Analysis

This project analyzes customer churn data to identify patterns that lead to customers leaving a service. The dataset was cleaned and processed using Pandas, and exploratory data analysis was performed to understand factors influencing customer retention and churn behavior.

### 2. Netflix Data Analysis

This project explores Netflix dataset information such as movie and TV show distribution, release years, genres, and content trends. Pandas operations were used to analyze the dataset and understand how streaming content has evolved over time.

### 3. Video Game Sales Analysis

This project analyzes video game sales data across different platforms and regions. Using Pandas, the dataset was processed to identify the most popular games, top-selling platforms, and regional sales trends in the gaming industry.

### 4. Weather Data Analysis

This project performs analysis on weather-related data to understand patterns in temperature, humidity, and other environmental factors. Data filtering, aggregation, and statistical summaries were used to examine weather trends and conditions.

### 5. Zomato Review Analysis

This project analyzes restaurant reviews and ratings from Zomato data. The dataset was explored to understand customer preferences, restaurant ratings distribution, and factors affecting customer satisfaction.

## Key Skills Demonstrated

* Data Cleaning and Preprocessing
* Handling Missing Values
* Data Filtering and Transformation
* Exploratory Data Analysis (EDA)
* Data Aggregation and GroupBy Operations
* Data Visualization

## Conclusion

These projects highlight the practical use of Pandas for analyzing real-world datasets and extracting valuable insights. They demonstrate essential data analysis workflows that are widely used in data science, business analytics, and machine learning projects.
